# Elearning-Platform
Extracurricular Elearning Platform with RoUPSKILL – Skill-Based E-Learning Platform

📌 Project Overview
UPSKILL is a web-based platform designed to support extracurricular skill learning. It enables instructors to upload tutorial videos and showcase handmade products, while learners can explore courses, watch videos, and contact instructors.
The platform bridges the gap between skill learning and small-scale entrepreneurship by integrating both learning and product showcasing in a single system.

🎯 Features
User registration and login
Video learning module for skill development
Product showcase module for handmade items
Instructor contact via email and phone
Secure password hashing
File upload for videos and images
Responsive web interface

🛠️ Technologies Used
Frontend

CSS
JavaScript
Backend

⚙️ System Workflow
Users register and log in to the platform
Users browse available skill courses
Users view course videos and details
System retrieves data from the database
Users explore related products
Users contact instructors for further interaction

🔐 Security Features
Password hashing using Werkzeug (PBKDF2)
Input validation for forms
Secure file upload handling
Session-based authentication
